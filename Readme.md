# Titanic Survival Prediction

Machine Learning project predicting passenger survival.

Dataset: Titanic Dataset (Kaggle)

Technologies:
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

Steps:
1. Data Cleaning
2. Handling Missing Values
3. Label Encoding
4. Logistic Regression Model
5. Model Evaluation

Accuracy: ~80%
